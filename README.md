## Featured Project: Network Monitoring & Analysis for AWS Cloud Security

This project focuses on detecting suspicious activity targeting AWS cloud users and protecting user data from unauthorized access attempts.

The workflow includes:

- Capturing and analyzing network packets using **Wireshark** and **TShark**
- Scanning and identifying suspicious hosts using **Nmap**
- Using **Python libraries** to parse network data and extract suspicious IP addresses
- Enriching suspicious IPs with geolocation details
- Identifying possible attacker behavior and unusual traffic patterns
- Supporting IP-based blocking to help protect AWS cloud resources and user data
- Documenting findings for incident response and cloud security improvement

This project reflects practical cybersecurity work across **network defense, cloud security, threat analysis, and incident response**.

## Featured Project: AI-Powered SOC Alert Triage System

This project aligns with the growing use of AI in modern cybersecurity operations.

The goal is to support analysts by improving alert context, reducing repetitive manual work, and helping prioritize alerts based on risk.

The workflow includes:

- Ingesting security event data
- Classifying alerts based on suspicious patterns
- Reducing false positives through contextual analysis
- Enriching alerts with indicators of compromise
- Mapping activity to MITRE ATT&CK where applicable
- Suggesting possible investigation and response actions
- Supporting faster and more consistent SOC decision-making

This project demonstrates how AI-assisted workflows can improve SOC efficiency while still relying on analyst judgment, evidence review, and risk-based thinking.
